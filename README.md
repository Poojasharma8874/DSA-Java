# DSA Practice Repository

This repository contains solutions for selected Data Structures & Algorithms (DSA) problems in Java.
Currently focused on Arrays and Stack problems, with techniques for duplicates removal, two-pointer method, and string handling.

## Topics Covered

### Arrays
- Remove duplicates from sorted arrays
  - Two-pointer technique
  - Using sorted array comparison
- String checks
  - Using `isBlank()` method

### Stack
- Basic stack problems (push, pop, peek)
- Example problems solved using Stack.

## How to Use

1. Clone this repository:
   git clone https://github.com/username/DSA-Practice.git

2. Navigate to the relevant topic folder and run Java files:
   cd Arrays
   javac RemoveDuplicates.java
   java RemoveDuplicates

3. You can add your own solutions in the appropriate folder.

## Notes
- All solutions are written in Java
- Problems are solved by taking steps for clarity and understanding.
- Follow folder structure to easily find and run problems.
